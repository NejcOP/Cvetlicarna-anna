# Cvetličarna ANNA

Butična cvetličarna v Sloveniji - Moderna spletna stran z galerijo in obrazcem za naročila.

## 🌸 Projekt

Elegantna in visoko-konvertirajoča spletna stran za cvetličarno ANNA, zgrajjena z HTML, CSS in JavaScript.

## ✨ Funkcionalnosti

- **Domača stran**: Predstavitev cvetličarne z Hero sekcijo, storitvami, galerijo in referencami
- **Galerija**: Dinamična galerija s filtri po kategorijah in lightbox funkcionalnostjo
- **Obrazec za naročila**: Popoln obrazec z validacijo za oddajo naročil
- **Mobilna navigacija**: Responsive hamburger meni za mobilne naprave
- **WhatsApp integracija**: Sticky gumb za hitro naročanje

## 🎨 Tehnologije

- HTML5
- CSS3 (Custom styling, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
- Lucide Icons

## 📱 Responsive Design

Optimizirano za vse naprave - od mobilnih telefonov do namiznih računalnikov.

## 🚀 Kako zagnati

Preprosto odprite `index.html` v brskalniku.

## 📄 Struktura projekta

```
├── index.html          # Domača stran
├── gallery.html        # Galerija
├── order.html          # Obrazec za naročila
├── styles.css          # Glavne stilske datoteke
├── gallery.css         # Stilske datoteke za galerijo
├── order.css           # Stilske datoteke za obrazec
├── navigation.js       # Navigacijska logika
├── gallery.js          # Galerijska logika
└── order.js            # Logika obrazca
```
